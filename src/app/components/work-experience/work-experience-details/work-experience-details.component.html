On Job Training es una aplicación desarrollada para el cliente Tenaris hace uso de servicios REST y SOAP (del lado cliente), desarrollado con Angular 8 del lado Front-End haciendo uso de múltiples componentes citados por rutas, con protecciones de las mismas por guardianes, se usaron múltiples librerías para el uso de estilos y validaciones como Bootstrap 4. Por otro lado, usa como Back-End la tecnología .NET Framework donde está separado por capas, unidad de negocio, controladores, configuración de servicios, etc. La base de datos está construida en SQL Server por tablas y stored procedure. Para el versionado del proyecto usamos Git como controlador de versiones y repositorio remoto incluimos Azure DevOps para gestion de tickets, code review, ramificación, deploy a distintos ambientes, etc.
